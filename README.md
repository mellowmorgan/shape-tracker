# Shape Tracker

#### Webpage for tracking types of triangles

#### By Morgan Waites

## Technologies Used

* Javascript
* HTML/CSS/Bootstrap

## Description

Enter 3 side lengths and it will display type of triangle.

## Setup/Installation Requirements

* git clone this repo
* navigate to top-level of directory
* npm install
* npm run start
